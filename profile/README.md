Some projects are released using a [Sponsorware](https://github.com/sponsorware/docs) model.

Sponsors receive **immediate access** to private repositories and early releases. Once a project reaches its published sponsor threshold, it becomes **free and open source** for everyone.

> **Early access funds the work. Open source shares the result.**

## Why?

Open source is valuable, but it takes time to build and maintain.

Sponsorware allows early adopters to fund development while guaranteeing that **successful projects eventually return to the community**.

When a project reaches its threshold:
- Repository becomes public
- MIT License applied
- v1.0.0 released
- Development continues in the open
